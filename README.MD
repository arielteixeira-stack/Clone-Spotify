# Clone do Spotify 

este é um projeto de clone spotify, desenvolvido para fins educacionais.

## Tecnologias Utilizadas  

- HTML 5 
- CSS 3
- javaScript
- Tailwind CSS  (Versão 4.x)
- Node.js (versão 24.x)

## Instalação 

1. clone este 

```bash

git clone https://github.com/arielteixeira-stack/Clone-Spotify.git
```
2. Navegue até o diretorio do projeto:

```bash
cd CloneSpotify
```

3. Instale as dependências: 

```bash 
npm install

```
